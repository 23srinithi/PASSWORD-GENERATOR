# Password Generator

## Description
This is a **Python Password Generator** that allows users to create strong and secure passwords easily. The program generates passwords containing **letters (uppercase & lowercase), numbers, and symbols** based on the length entered by the user.  

It is a beginner-friendly project aimed at **learning Python basics, user input, loops, and module usage**.

## Features
- Generate random passwords of user-defined length  
- Includes uppercase & lowercase letters, numbers, and symbols  
- Simple and interactive terminal interface  
- Quick password generation for personal use
